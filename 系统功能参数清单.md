# 知识图谱与学习成效分析系统 · 功能参数清单

> 当前系统版本：知识图谱与学习成效分析演示原型（面向国网电力装表接电岗位）
> 技术栈：Next.js 16 (App Router) + React 19 + TypeScript + Tailwind CSS 4 + shadcn/ui
> 数据模式：全 mock 数据客户端渲染，无后端 API
> 用户角色：学员（student）、培训管理员（admin）

---

## 一、总体架构

| 参数项 | 当前实现 |
|--------|----------|
| 总体定位 | 知识图谱驱动的岗位能力诊断与学习成效分析平台，面向电网企业装表接电岗位培训 |
| 用户规模 | 5 名学生（mock）、管理员视图 |
| 角色体系 | 学员、培训管理员双角色，通过 URL 参数 `?role=` 切换 |
| 统一架构 | 单页面应用，AppShell 侧边栏导航，角色感知菜单 |
| 数据模型 | Position（岗位）→ CapabilityDomain（能力领域）→ CapabilityUnit（能力单元）→ KnowledgePoint（知识点）→ Course（课程）五层模型 |
| 扩展接口 | 前端模型管理支持动态新增岗位/领域/单元/知识点/课程 |

## 三、学员端功能参数

### 3.1 AI 工作台（`/home`）

| 参数项 | 当前实现 |
|--------|----------|
| 数据看板 | 完成工单数、学习时长、已完成课程数、岗位排名、平均作业时长、工单合格率 6 项统计 |
| AI 智识问答 | 快捷入口：智识问答、图像智询、视频智询、课程智询 |
| 快捷建议 | 多行智能学习建议列表 |
| 个人 Tab | AI 工作台（默认） |
| 学习 Tab | 成效分析、知识图谱、学习推荐、学习评测 |
| 侧边栏 | 直播课堂、个人课堂、实训仿真、课程智作、AI 工作台、成效分析、知识图谱、学习推荐、学习评测 |
| 时段筛选 | 近一周/近一月/近一年 |

### 3.2 执行记录（`/history`）

| 参数项 | 当前实现 |
|--------|----------|
| 数据源 | 模拟工单系统自动同步（只读展示） |
| 数据层级 | 记录 → 任务 → 任务分类 → 问题类型 → 具体问题 → 扣分项（六级层级） |
| 统计看板 | 执行记录数、累计扣分、问题项数量 |
| 记录列表 | 按日期排序展示执行记录，含能力领域名称、关联能力单元、扣分影响 |
| 任务明细 | 可展开查看每个记录的任务层级、问题类型、扣分项明细 |
| 统计分析 | 按任务分类统计、按问题类型统计、按五维维度汇总、按扣分项排序 Top 8 |
| 五维维度 | 数量、质量、安全、规范、效率 |

### 3.3 五维能力诊断（`/diagnosis`）

| 参数项 | 当前实现 |
|--------|----------|
| 时间范围筛选 | 起始日期/结束日期选择器 |
| 统计卡片 | 累计扣分、薄弱能力单元数、综合健康分 |
| 五维雷达图 | 基于扣分数据的负向计分雷达图（Recharts 实现） |
| 维度详情 | 五个维度分别展开，展示扣分明细（问题类型、问题名称、扣分项、日期） |
| 薄弱能力单元 | 按扣分排序展示薄弱单元，支持导入知识图谱高亮 |
| 诊断逻辑 | 基于具体扣分项明细数据定位短板，而非雷达图整体分值 |

### 3.4 知识图谱（`/knowledge-graph`）

| 参数项 | 当前实现 |
|--------|----------|
| 图谱视图 | 静态分层视图（React Flow）和力导向视图（D3.js）双模式切换 |
| 图谱层级 | Position（岗位）→ Domain（能力领域）→ Unit（能力单元）→ Knowledge（知识点）→ Course（课程） |
| 数据来源 | 学生端加载个人图谱，管理员加载全量图谱 |
| 高亮标注 | 支持从诊断页面导入薄弱能力单元高亮显示 |
| 节点交互 | 节点悬停/点击详情、拖拽、缩放 |
| 节点详情面板 | 展示关联能力单元、知识点、课程信息 |
| 时间筛选 | 起始/结束日期筛选 |
| 角色适配 | 学员视图高亮个人薄弱节点；管理员视图展示全量 |

### 3.5 学习推荐（`/learning`）

| 参数项 | 当前实现 |
|--------|----------|
| 能力单元列表 | 按掌握度排序（未达标→达标→超基准），展示当前分数与基准线 |
| 能力图谱 | 左侧能力单元列表 + 中间能力图谱 + 右侧评价详情三栏布局 |
| 能力详情 | 掌握度概览卡片（分数/基准/条形图）、五维评分数据 Tab、历史执行记录表 |
| 学习建议 | AI 诊断建议（达标/未达标） |
| 推荐学习资源 | 按学习路径分组（安全风险专题、理论知识、技能教学、三维仿真） |
| 资源类型 | 视频课程、课件、试题，支持外链跳转学习平台 |
| 图谱视图切换 | 静态分层/力导向双模式 |

### 3.6 学习评测（`/assessment`）

| 参数项 | 当前实现 |
|--------|----------|
| 统计卡片 | 已完成课程数、已完成考试数、考试通过率 |
| AI 诊断 | 薄弱能力单元诊断 + 原因说明 + 推荐课程 |
| 随堂测试 | 弹窗模式，逐题作答（单选），支持上下题导航、提交评分 |
| 考试记录 | 表格展示考试名称、得分、通过状态、日期 |
| 学习记录 | 列表展示已学课程及日期 |
| 通过判定 | 得分 ≥ 60% 判定为通过 |

### 3.7 成效分析（`/profile`）

| 参数项 | 当前实现 |
|--------|----------|
| 能力概览 | 综合均分、能力达标率、达标能力单元数、待提升项 |
| 岗位能力模型 | 全部能力单元列表，展示掌握度条形图，支持点击查看历史变化 |
| 能力变化历史弹窗 | 折线图展示能力随时间变化曲线 + 升降原因明细（学习/执行/衰减） |
| 学习成效对比 | 各能力单元学习前后达标等级对比（未达标→达标→超基准） |
| 五维能力提升 | 柱状图展示学习前后五维分值对比 |
| 得分变化曲线 | 多线折线图展示各能力单元得分随时间变化 |
| 工单明细 | 可展开查看执行记录任务层级扣分明细 |
| 闭环提示 | "执行驱动诊断 → 知识图谱关联 → 智能学习推荐 → 成效提升" 闭环 |

## 五、数据模型参数

### 5.1 数据类型

| 类型 | 字段 | 说明 |
|------|------|------|
| Position | id, name, code | 岗位基本信息 |
| CapabilityDomain | id, name, code, positionId, unitIds[], taskDescription, standardDuration, riskDeductionRules, scoreRules, type, batchGroup, creator, createTime, version | 能力领域（工单模板） |
| CapabilityUnit | id, name, code, domainId, description, baseline, knowledgeIds[], assessmentConfig | 能力单元 |
| AssessmentConfig | workload[], quality[], efficiency[], compliance[], safety[] | 五维考核标准配置 |
| KnowledgePoint | id, name, code, level, parentId, abilityIds[], capabilityUnitIds[], courseIds[] | 知识点（支持多级树） |
| Course | id, title, type, duration, knowledgeIds[], parentId, description | 课程/课件/测验 |
| Student | id, name, positionIds[], department, joinDate | 学生 |
| StudentAbilityScore | studentId, positionId, abilityId, date, score, source | 学生能力单元日分数 |
| StudentWorkorderRecord | studentId, workorderId, date, dimensionScore, abilityId, scoreImpact, tasks[] | 执行记录 |
| StudentLearningRecord | studentId, courseId, date, score | 学习记录 |
| Quiz | id, courseId, questions[] | 随堂测验 |
| QuizQuestion | id, question, options[], answer | 测验题目 |
| Recommendation | id, studentId, abilityId, reason, courseIds[], createdDate, status | 学习推荐 |
| EffectivenessReport | abilityId, abilityName, before, after, dimensionDeltas | 成效评估报告 |
| GraphNode | id, label, type, level | 图谱节点（5 种类型） |
| GraphEdge | source, target | 图谱边 |
| WorkorderSyncRecord | id, domainId, date, studentName, department, dimensionScores, status | 工单同步记录 |
